# Automates finis (Python)

Projet de mathématiques appliquées consistant à traduire en **Python** les principales transformations sur automates finis, à partir du formalisme vu en cours, puis à valider les résultats sur des exemples.

## Objectif
- Passer d’un algorithme mathématiques à une implémentation claire et testable.
- Simplifier et vérifier des automates (cohérence, équivalence de langage selon les transformations).

## Fonctionnalités
- Manipulation d’automates (états, alphabet, transitions)
- Transformations sur automates (ex. complétion, déterminisation, standardisation, minimisation selon le cours)
- Jeux de tests et validation sur cas d’usage

## Technologies
Python

```bash
python main.py
